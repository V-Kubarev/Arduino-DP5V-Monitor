# Архитектура проекта DP-5V Arduino Geiger Counter

## 1. Общий обзор

Проект DP-5V — микроконтроллерная система измерения и индикации
ионизирующего излучения на основе счётчика Гейгера–Мюллера.

Система включает:
- аппаратную часть (датчик, питание, индикация);
- программную часть (прошивка Arduino);
- алгоритмы статистической обработки измерений.

---

## 2. Аппаратная архитектура

### 2.1 Основные компоненты

| Компонент | Назначение |
|----------|-----------|
| Счётчик Гейгера | Регистрация ионизирующих событий |
| HV-модуль | Питание ГМ-трубки |
| Arduino MCU | Обработка импульсов |
| LCD 16×2 (I2C) | Отображение данных |
| Зуммер | Звуковая индикация |
| Кнопка | Управление режимами |

---

### 2.2 Подключение периферии

| Сигнал | Пин MCU | Описание |
|-------|--------|----------|
| Geiger pulse | D2 (INT) | Импульсы от трубки |
| Button | D3 | Пользовательский ввод |
| Buzzer | D7 | Звуковая индикация |
| LCD | I2C | Адрес 0x27 |

---

## 3. Программная архитектура

Прошивка построена по событийно-циклической модели:
- регистрация импульсов — в прерывании;
- обработка и интерфейс — в основном цикле.

Основные логические блоки:
- регистрация импульсов;
- статистическая обработка;
- расчёт дозы;
- пользовательский интерфейс.
